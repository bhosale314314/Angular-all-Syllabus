
Intercomponent Communication using Service:

Service is used for intercomponent communication with the help of special operators from RxJs.

Special operators are those operators who are having storing capability.

1. subject: It is special multi cast operator from RxJs which can hold the current data.

2. Behaviour Subject: It is special multi cast operator from RxJs which can hold the current data as well as previous data.



Third Party Library Installation: 

prime Ng:
	go to : https://www.primefaces.org/primeng/setup

	npm install primeicons --save
